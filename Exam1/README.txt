Instruções para compilação:
    - Executar o comando "make" em um terminal aberto neste diretorio

    - O arquivo executavel será gerado neste diretorio com o nome "main"

    - Case necessite de recompilação, basta executar o comando "make" novamente,
        o mesmo recompilará os arquivos que forem atualizados

    - O comando "make clean" remove todos os aqrquivos gerados na compilação
        possibilitando um compilação limpa
